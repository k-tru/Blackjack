Blackjack Casino Game

A lightweight, browser-based Blackjack game built with vanilla HTML, CSS, and JavaScript.

Features
* **Automated Dealer:** Hits automatically until reaching a score of 17.
* **Bankroll System:** Starts at \$200 with a \$20 entry fee and automatic payouts.
* **Responsive Layout:** Clean, simple casino-table styling.

How to Play
1. Open `index.html` in any browser.
2. Click **Start** to buy in for \$20.
3. Click **New Card** to draw, or **Stand** to pass to the dealer.
